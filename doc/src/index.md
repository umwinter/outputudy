# プロジェクトコンテキスト

このプロジェクトは、フロントエンド学習のためのアプリケーションです。

## 構成

### Frontend
- **Framework**: Next.js (App Router)
- **UI Library**: shadcn/ui
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Version Management**: mise (Node.js)

### Backend
- **Framework**: FastAPI
- **Language**: Python 3.12+
- **Environment Management**: venv (.venv)
- **Dependency Management**: requirements.txt

## 目的
本プロジェクトは、モダンなWeb開発技術（Next.js, FastAPI）を用いた学習と実践を目的としています。
